# Finder 右键扩展工具

**中文** | [English](#english-version)

---

## 功能特性

### 1. 右键菜单增强
在访达（Finder）右键菜单中新增以下创建选项：

#### 文本文件
- **新建纯文本文件** (.txt)
- **新建富文本文件** (.rtf)

#### 二进制文件
- **新建 Word 文档** (.docx)
- **新建 PowerPoint 演示文稿** (.pptx)
- **新建 Excel 表格** (.xlsx)
- **新建 PDF 文档** (.pdf)

### 2. 快捷操作
- **双击 Shift 键**：快速打开剪贴板历史记录面板

## 安装说明
1. 下载最新版本安装包
2. 打开 `.dmg` 文件
3. 将应用拖拽到“应用程序”文件夹
4. 首次运行时需要在系统设置中授予辅助功能权限

## 使用方法
### 创建新文件
1. 在访达窗口或桌面空白处右键点击
2. 选择“新建”菜单
3. 选择需要创建的文件类型

### 剪贴板历史
- 在任何界面双击 **Shift 键**（左右 Shift 均可）
- 从历史记录中选择内容即可粘贴

## 常见问题
**Q：右键菜单没有显示新选项？**  
A：请检查应用是否已启动，并在系统设置 > 隐私与安全性 > 辅助功能中已授权。

**Q：双击 Shift 无效？**  
A：请确保应用正在运行，且快捷键未被其他应用占用。

# Finder Context Menu Extension

**English** | [中文](#)

## Features

### 1. Enhanced Context Menu
Adds the following options to Finder's right-click menu:

#### Text Files
- **New Plain Text File** (.txt)
- **New Rich Text File** (.rtf)

#### Binary Files
- **New Word Document** (.docx)
- **New PowerPoint Presentation** (.pptx)
- **New Excel Spreadsheet** (.xlsx)
- **New PDF Document** (.pdf)

### 2. Quick Actions
- **Double-tap Shift Key**: Open clipboard history panel

## Installation
1. Download the latest installation package
2. Open the `.dmg` file
3. Drag the app to "Applications" folder
4. Grant accessibility permissions in System Settings on first launch

## Usage
### Create New Files
1. Right-click in Finder window or on desktop
2. Select "New" menu
3. Choose desired file type

### Clipboard History
- Double-tap **Shift key** (left or right) in any application
- Select content from history to paste

---

## FAQ
**Q：New options not showing in right-click menu？**  
A：Ensure the app is running and authorized in System Settings > Privacy & Security > Accessibility.

**Q：Double-tap Shift not working？**  
A：Make sure the app is running and the shortcut is not occupied by other applications.
